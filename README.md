HOW TO RUN:

1. Run MySQL 5.7<br/>
If you have docker and docker-compose installed, run from project root<br/> 
docker-compose up<br/>

Or run your own mysql with the settings stated in docker-compose.yml 

2. JDK8 and Gradle
From server/ run:
./gradlew run

3. ReactJS

From client/ run:
npm install && npm start
